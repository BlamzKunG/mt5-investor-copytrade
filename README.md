# mt5-investor-copytrade

โปรแกรมก๊อปปี้เทรดทองคำ (XAUUSD) จากพอร์ต Master (อ่านด้วยรหัสผ่าน investor) ไปยังพอร์ต Client ทำงานผ่านตัวกลาง Redis เพื่อให้รับส่งข้อมูลคำสั่งเปิดปิดออเดอร์ได้รวดเร็วแบบเรียลไทม์ และควบคุมทุกอย่างได้ผ่านโปรแกรมหน้าต่าง GUI ในหน้าเดียว

## โครงสร้างระบบ
ระบบจะมีไฟล์ทำงานหลักอยู่ 3 ส่วนคือ
1. master.py คอยจับตาดูออเดอร์ในพอร์ต Master ทุกๆ เสี้ยววินาที แล้วส่งคำสั่งไปที่ Redis
2. client.py รอรับสัญญาณจาก Redis เพื่อนำมาสั่งออกออเดอร์หรือเปิดปิดในพอร์ต Client
3. dashboard.py หน้าต่างควบคุมแบบ GUI ที่เขียนด้วยภาษาไธทอน เพื่อให้กดเซฟการตั้งค่าและกดเริ่มหยุดโปรแกรมได้ง่ายๆ

## การตั้งค่าใช้งาน (config.json)
หลังจากรันโปรแกรมครั้งแรก ตัวระบบจะสร้างไฟล์ config.json ขึ้นมาให้โดยอัตโนมัติเพื่อใช้ควบคุมการทำงาน คุณสามารถกรอกพาธของโปรแกรมเทรดหรือตั้งค่าการก็อปปี้ได้จากหน้านี้หรือบนหน้าต่างควบคุมโดยตรง

{
    "redis": {
        "host": "localhost",
        "port": 6379,
        "db": 0,
        "channel": "copy_trade_gold"
    },
    "master": {
        "terminal_path": "ตำแหน่งของโปรแกรม terminal64.exe ฝั่ง Master",
        "poll_interval_seconds": 0.05
    },
    "client": {
        "terminal_path": "ตำแหน่งของโปรแกรม terminal64.exe ฝั่ง Client",
        "gold_symbol": "สัญลักษณ์ทองของฝั่ง Client เช่น XAUUSD หรือ XAUUSD-ECN",
        "lot_multiplier": 0.5,
        "lot_minimum": 0.01,
        "magic_number": 999999,
        "deviation": 20
    }
}

คำอธิบายหัวข้อที่สำคัญ
- lot_multiplier คืออัตราคูณล็อต เช่น ตั้งเป็น 0.5 ขนาดล็อตฝั่ง Client จะเป็นครึ่งหนึ่งของ Master เสมอ
- poll_interval_seconds คือเวลาในการเข้าไปตรวจเช็คฝั่ง Master (ยิ่งตัวเลขน้อย ยิ่งตามก๊อปปี้ได้ไวขึ้น)

## วิธีเปิดใช้งาน
1. ตรวจสอบให้แน่ใจว่าติดตั้ง Python และมีโปรแกรม Redis ทำงานอยู่ในเครื่องคอมพิวเตอร์
2. เปิดโปรแกรมหน้าจอควบคุมด้วยคำสั่ง
python dashboard.py
3. ระบุตำแหน่งพาธของโปรแกรม MT5 (.exe) ของทั้งสองพอร์ตให้ถูกต้องแล้วกดปุ่มบันทึก
4. กดปุ่มเริ่มทำงานเพื่อเปิดให้ระบบจับคู่และคัดลอกพอร์ตโดยอัตโนมัติ
